What is signals

Counter value without signals: {{counter}}

This works because Angular use library Zone js for change detection, which is patching all the DOM API(setTimeout, AJAX requests etc.). It's checking all component's expressions before an event and after an event. It's marked this value dirty and should be checked after each action.

Counter value like a signal: {{counterSignal()}}

Signal must always have a value ( signal(0 / null / some value)). Now this is WritableSignal of type number. To show value in the template, it should be called is double squire brackets counterSignal().
For change value of the signal we should use this syntax: this.counterSignal.set(this.counterSignal()+1)
